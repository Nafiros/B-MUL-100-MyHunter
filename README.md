## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

You just need to install CSFML binding :)
https://www.sfml-dev.org/download/csfml/

### Installation

1. Clone the repo
```sh
git clone git@github.com:Nafiros/B-DOP-500-Popeye.git
```
2. Build the project
```sh
make
```
